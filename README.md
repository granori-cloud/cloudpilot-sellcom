# CloudPilot AI — Sellcom Solutions

Landing page corporativa de **CloudPilot AI**, la plataforma de optimización de infraestructura multicloud con inteligencia artificial desarrollada por **Sellcom Solutions México**.

## 🚀 Descripción

Landing page de alto impacto para comunicar el valor de CloudPilot AI a equipos ejecutivos, de TI y financieros. Incluye:

- Dashboard mockup interactivo en CSS puro
- Análisis de desperdicio en nube con gráficos CSS
- 6 funcionalidades clave con animaciones hover
- 6 reportes ejecutivos descritos
- Flujo de 4 pasos "Cómo funciona"
- 3 casos de uso por perfil ejecutivo
- Sección de métricas de impacto
- Footer con enlaces a Sellcom Solutions
